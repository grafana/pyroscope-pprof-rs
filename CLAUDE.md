# Claude Instructions for pprof-rs

## Pull Request Titles

Use conventional commit format: `<type>: <description>`

Allowed types:
- `feat:` - new feature
- `fix:` - bug fix
- `perf:` - performance improvement
- `docs:` - documentation
- `refactor:` - code refactoring
- `test:` - tests
- `ci:` - CI/CD changes
- `chore:` - maintenance
- `build:` - build system
- `style:` - code style
- `revert:` - revert changes
- Add `!` for breaking changes: `feat!:`, `fix!:`
